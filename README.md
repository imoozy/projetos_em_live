# Projetos em Live

Este repositório está sendo aprimorado diariamente com estudos "do 0 ao avançado" de cada linguagem que vejo na faculdade. Aqui eu adiciono conteúdos extras, exercícios e incrementos práticos sempre que aprendo algo novo.

O objetivo é evoluir constantemente este conjunto de projetos para, no futuro, torná-lo um portfólio sólido e representativo das minhas habilidades.

- Atualizações: diárias, conforme avanço nas matérias.
- Conteúdo: exemplos, exercícios, projetos pequenos e anotações.
- Estrutura: pastas organizadas por linguagem/tecnologia.

Contribuições e sugestões são bem-vindas!
